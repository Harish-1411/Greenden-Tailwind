# 🌿 Greenden Website

A modern and responsive plant shop website built using **HTML** and **Tailwind CSS**.
This project showcases a clean UI design for an online plant store.

## 📌 Project Description

Greenden is a simple and elegant website designed for a plant-selling business.
It includes a responsive layout, product sections, navigation bar, and footer.

This project was created to practice:

* HTML structure
* Tailwind CSS styling
* Responsive web design
* Clean UI layout
